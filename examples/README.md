# [JEPROF](https://djoezeke.github.io/jeprof) [ EXAMPLES ]

This folder contains code examples for the C and C++ APIs.

> [!NOTE]
> Jeprof is still in development, **current version is incomplete and nonfunctional**.
> There is more room for improvements.

The purpose of Examples is to showcase integration, let you try Jeprof, and guide you toward
integrating Jeprof in your own application.

## Getting Started

Integration in a typical existing application, should take no time.

```cpp
#include <Jeprof/Jeprof.h>
```

Main resource:

- Read **[Documentation](https://djoezeke.github.io/jeprof/)** for detailed on how to integrate Jeprof.
